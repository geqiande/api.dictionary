# Humanities 人文相关 api

### today in History 历史上的今天
* Language `zh_cn`
* area `zh_cn`
* Url `http://www.todayonhistory.com/index.php`
* Type `POST` `GET`
* Jsonp `No`
* Description
  ```html
    en: Get what happened today in history
    cn: 获取历史上今天所发生的事情
  ```
* Params
  ```javascript
    {
      "m": "content", // default
      "c": "index", // default
      "a": "json_event", // default
      "page": 0, // page number
      "pagesize": 40, // page size
      "month": 10, // month
      "day": 16 // day
    }
  ```
