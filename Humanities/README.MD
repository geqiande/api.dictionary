# Humanities 人文相关 api

### today in History 历史上的今天
* Language `zh_cn`
* Url `http://www.todayonhistory.com/index.php`
* Type `POST`
* Description
  ```html
    en: Get what happened today in history
    cn: 获取历史上今天所发生的事情
  ```
* Params 
  ```json
  {
    "m": "content",
    "c": "index",
    "a": "json_event",
    "page": 0,
    "pagesize": 40,
    "month": 10,
    "day": 16
  }
  ```
